# Undertale-fangame-Justtale-beta-1

## Запуск игры
1. Скачать файлы
2. Запустить Justtale_beta_1.exe
3. В Justtake_beta_1.rar находятся файлы проекта для Gamemaker studio 2. Среда выполнения: 2024.6.2.162

## Управление:
- Меню: управление мышкой
- Otherworld:
  - передвижение с помощью стрелок
  - Взаимодействие/следующий диалог/выбор с помощью кнопки Z
  - Отмена с помощью кнопки X
  - Открытие/закрытие инвенторя с помощью кнопки C
- Бой(меню):
  - передвижение с помощью стрелок
  - Взаимодействие/следующий диалог/выбор с помощью кнопки Z
  - Отмена с помощью кнопки X
- Бой(атака):
  - передвижение с помощью стрелок
  - Во время нажатия кнопки X скорость передвижения уменьшается

## Как играть?
- Otherworld
- - Можно изучить мир и 2 комнаты, взаимодействовать с объектами или пойти дальше и вступить в битву
- Битва
- - Два варианта развития событий: бой или пощада
